## Improved

- [DAS AI placement]: DAS carrier channel width is now selectable from the band's allowed widths instead of being fixed at 20 MHz.
- [DAS — Everon 6200]: Digital distributor now displays as "Switch" and port counts are editable on placed instances.
- [Walk-test]: PCI is now a first-class heatmap KPI with categorical dominance rendering and a per-KPI sample table with CSV export.
- [Walk-test]: Switching to "Site survey data" no longer forces an upload prompt when survey data already exists for the layout.
- [AP/RU placement defaults]: Mounting type and height now pre-fill based on project IO type and layout ceiling height.
- [Riser diagram]: EIRP is now shown on the riser diagram for WiFi and LoRa, not only DAS.
- [DAS radiated power]: Active DAS antennas now simulate at RU power rather than headend power.
- [Vector PDF wall parsing]: Multi-floor digital-twin wall parsing is significantly faster on dense CAD exports.
- [Multi-floor projects]: Project load time is reduced on layouts with large wall counts across multiple floors.

## Fixed

- Fixed DAS cabling visibility toggle having no effect on generated report heatmaps.
- Fixed cellular AP config Mounting, Height, and Pin-to-story edits being lost on save.
- Fixed geolocation wizard "Next" button submitting before the map had finished loading.
- Fixed walk-test PCI and EARFCN heatmap selection crashing and showing a black canvas.
- Fixed project settings channel selectors rendering oversized for CBRS and C-Band rows.
- Fixed walk-test site-survey heatmap remaining visible after switching to the eino walk-test view.
- Fixed RSSI missing from the walk-test KPI resolver.
